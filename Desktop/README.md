# Ensure you have Yarn installed:

```bash
cd Desktop
corepack enable
```

# If on Linux

```bash
sudo npm i --force
```

# Ensure you have the correct version of yarn:

```bash
yarn set version stable
```

# install dependancies: (Use sudo if on Linux)

```bash
yarn install
```

# to run rollup: 

```bash
yarn run rollup -c -w
```

# To start electron: (in new terminal):

```bash
yarn electron .
```
